# Ecommerce Sales Analysis Dashboard

## Project Overview

This project analyzes ecommerce sales data using Python and Power BI to identify sales trends, top-performing products, and regional performance.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Power BI

## Project Steps

1. Data Cleaning and Preprocessing using Python
2. Exploratory Data Analysis (EDA)
3. Visualization of sales trends
4. Building an interactive Power BI dashboard

## Key Insights

* Technology category generated the highest sales.
* West region contributed the most revenue.
* Top 10 products account for a large portion of sales.
* Sales show monthly fluctuations across the year.

## Dashboard Features

* Total Sales KPI
* Sales by Category
* Monthly Sales Trend
* Sales by Region
* Top 10 Products by Sales

## Project Structure

dataset/ – contains raw sales data
python_analysis/ – Jupyter Notebook for analysis
powerbi_dashboard/ – Power BI dashboard file
